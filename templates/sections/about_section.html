Tentang Aplikasi

LearnStyle Expert System

LearnStyle merupakan sistem pakar berbasis web yang dirancang untuk membantu siswa mengenali gaya belajar dominan mereka melalui assessment interaktif modern berbasis aturan (rule-based system).

🧠

Sistem Pakar

Menggunakan metode rule-based inference untuk menentukan gaya belajar dominan siswa berdasarkan jawaban assessment.

📊

Analytics Dashboard

Menampilkan visualisasi data, diagram, dan insight gaya belajar siswa untuk membantu guru BK dan sekolah.

🎯

Personalized Recommendation

Memberikan rekomendasi belajar untuk siswa, guru, dan orang tua berdasarkan hasil assessment.

Tujuan Aplikasi

Membantu siswa memahami cara belajar paling efektif, membantu guru menentukan strategi pembelajaran, serta membantu orang tua memahami kebutuhan belajar anak.

Teknologi yang Digunakan

  • Flask Framework
  • PostgreSQL Database
  • Docker Container
  • Chart.js Visualization
  • Bootstrap Responsive UI
